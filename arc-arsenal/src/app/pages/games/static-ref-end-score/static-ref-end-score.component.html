Volée de référence    

Description du jeu

Paramétrez le nombre de flèches par volée, le nombre de volées et la valeur de la volée de référence.

La volée de référence reste fixe pendant tout le jeu. À chaque volée, un nombre de points est attribué selon l'écart avec la référence :

  • volée ≥ réf + 4 → +2 pts
  • réf + 2 ≤ volée ≤ réf + 3 → +1 pt
  • réf - 1 ≤ volée ≤ réf + 1 → 0 pt
  • réf - 3 ≤ volée ≤ réf - 2 → -1 pt
  • volée ≤ réf - 4 → -2 pts

Le but est de terminer avec le total de points le plus élevé possible.
